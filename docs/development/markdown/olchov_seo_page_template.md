# olchov.uz — SEO Page Template (Per Conversion Pair)

Ushbu hujjat **olchov.uz** loyihasi uchun har bir konvertatsiya sahifasi  
(misol: `/uzunlik/dyuym-sm`) uchun **SEO Page Template** ni belgilaydi.

Bu template O‘zbekiston foydalanuvchilarining **real qidiruv xatti-harakati**  
(typo, sinonim, noto‘g‘ri yozilishlar) ni hisobga oladi.

---

## Asosiy SEO Strategiya

### Muhim kuzatuv
O‘zbek foydalanuvchilar ko‘pincha birliklarni turlicha yozadi:

- metr / metir / miter  
- santimetr / santimeter / sm  
- dyuym / dyum  
- selsiy / selsi / celsius  

### Qaror
- **URL** → faqat to‘g‘ri yozuv
- **Content** → to‘g‘ri + noto‘g‘ri variantlar

## H2 Strukturasi

### H2 #1 — Konvertor
```
Dyuym → Santimetr konvertori
```

Bu yerda:
- input
- real-time natija

---

### H2 #2 — Formula
```
Dyuymdan santimetrga o‘tkazish formulasi
```

Matn:
```
1 dyuym = 2.54 santimetr.
Santimetrni hisoblash uchun dyuym qiymatini 2.54 ga ko‘paytiring.
```

---

### H2 #3 — Yozilish variantlari (SEO gold)
```
Dyuym, dyum va boshqa yozilishlar haqida
```

Matn:
```
Ko‘pchilik foydalanuvchilar “dyum” yoki “dyuym” deb yozadi.
Ularning barchasi bir xil uzunlik birligini anglatadi.
```

---

### H2 #4 — Bog‘liq konvertorlar
```
Bog‘liq uzunlik konvertorlari
```

- santimetr → dyuym  
- metr → fut  
- millimetr → dyuym
