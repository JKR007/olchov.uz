# olchov.uz — V1 Information Architecture (IA)

Bu hujjat **olchov.uz** loyihasi uchun **V1 Information Architecture (IA)** ni yakuniy va tasdiqlangan holatda hujjatlashtiradi.

---

## IA Prinsiplari (V1)

### Kiritiladi
- O‘zbekistonda real ishlatiladigan birliklar
- Kundalik hayot, qurilish, texnika, ta’lim
- Google’da qidiriladigan konvertatsiyalar
- Har bir konvertatsiya jufti uchun alohida sahifa

### Kiritilmaydi (V1)
- Murakkab muhandislik birliklari
- Ilmiy / akademik konvertorlar
- Juda kam ishlatiladigan birliklar
- Valyuta (currency)
